
# 📄 AI Resume Screening System

An AI-powered Resume Screening System using Machine Learning, NLP and Streamlit.

The application analyzes resumes, extracts skills, predicts job category and calculates resume-job matching score.

---

## 🚀 Live Demo

🔗 **Application:**

https://airesumescreeningsystem-yedss29py8sdsttobgu52x.streamlit.app

---

## Features

✅ Upload multiple resumes  
✅ PDF and TXT support  
✅ Resume classification  
✅ Skill extraction  
✅ Job description matching  
✅ Resume match score  
✅ Machine Learning prediction  
✅ Interactive Streamlit UI

## How To Run

Clone:

```bash
git clone https://github.com/husnamea-it/AI_Resume_Screening_System.git
```

Install:

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
